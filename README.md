## в—Ҳ Overview

**forge.studio** is a single-page business portfolio built entirely without a build pipeline. React 18 and Babel run via CDN вҖ” the entire app lives in two files you can drag onto Netlify and ship today.

Designed for Indian local business owners who need to look world-class online. Dark aesthetic. Sharp typography. Conversion-focused layout.

## в—Ҳ Review Form вҖ” EmailJS Setup

The **Reviews** section collects visitor feedback and fires it straight to your inbox. Free tier covers 200 emails/month.

<br/>

**Step 1 вҖ” Sign up**

вҶ’ [emailjs.com](https://www.emailjs.com) В· Create a free account

<br/>

**Step 2 вҖ” Connect your email**

Dashboard вҶ’ *Email Services* вҶ’ *Add New Service* вҶ’ connect Gmail
Copy your **`SERVICE_ID`**

<br/>

**Step 3 вҖ” Create a template**

Dashboard вҶ’ *Email Templates* вҶ’ *Create New Template*

Use these variables in the template body:

```
Reviewer:  {{from_name}}
Business:  {{from_business}}
Rating:    {{rating}}
Review:    {{message}}
```

Copy your **`TEMPLATE_ID`**

<br/>

**Step 4 вҖ” Get your Public Key**

*Account* вҶ’ *API Keys* вҶ’ copy **`PUBLIC_KEY`**

<br/>

**Step 5 вҖ” Paste into the code**

Find the `handleSubmit` function inside `Testimonials()` in `portfolio.jsx`:

```js
body: JSON.stringify({
  service_id:  "YOUR_SERVICE_ID",    // вҶҗ replace
  template_id: "YOUR_TEMPLATE_ID",   // вҶҗ replace
  user_id:     "YOUR_PUBLIC_KEY",    // вҶҗ replace
  ...
})
```

вң… &nbsp;Done. Every star rating + review text lands in your inbox instantly.
